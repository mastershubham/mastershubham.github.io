What is this Project World Champion?

When I was child, I was fascinated by learning anything new. Despite going through a school curriculum, I was able to carry with me the same curiousity, passion and childishness. There is always so much to learn but instead of this fact making us overwhelmed, we can continue to accrue small gains. I think that by having a deeper understanding of reality we can make useful contributions and create value. If you have interesting ideas, consider helping me by sharing it.
